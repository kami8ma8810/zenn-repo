## 日本語説明文

### 短い説明（manifest.json から自動取得）

> X(Twitter)のポストをObsidianに保存します

### 詳細説明

```
X Clipper for Obsidian は、X（Twitter）のポストを Obsidian のノートとして保存する拡張機能です。

■ 主な機能
・ポストをMarkdown形式で永続保存
・添付画像を自動でローカルにダウンロード
・スレッド（連投リプライ）を1つのノートにまとめて保存
・引用ポストの内容も含めて保存
・タグ付けで整理可能

■ なぜこの拡張機能？
X のブックマークやいいねは、元のポストが削除されると消えてしまいます。
この拡張機能を使えば、大切なポストをあなたの Obsidian に永久保存できます。

■ 必要な環境
・Obsidian（https://obsidian.md/）
・Obsidian Local REST API プラグイン

■ プライバシーについて
・保存したデータはすべてローカルに保存されます
・外部サーバーへのデータ送信は一切ありません
・詳細はプライバシーポリシーをご確認ください
```

## English Description

### Short Description

> Save X(Twitter) posts to Obsidian with ease

### Full Description

```
X Clipper for Obsidian lets you save X (Twitter) posts as Obsidian notes.

■ Key Features
• Save posts in Markdown format for permanent storage
• Automatically download attached images locally
• Save threads (consecutive replies) as a single note
• Include quoted post content
• Organize with tags

■ Why This Extension?
X bookmarks and likes disappear when the original post is deleted.
With this extension, you can permanently save important posts to your Obsidian vault.

■ Requirements
• Obsidian (https://obsidian.md/)
• Obsidian Local REST API plugin

■ Privacy
• All saved data is stored locally
• No data is ever sent to external servers
• See our Privacy Policy for details
```
